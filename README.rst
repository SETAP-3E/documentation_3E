Budgetting App Documentation
============================

A full-stack budgeting application designed to help users track their accounts, budgets, and transactions efficiently.
Built with Flutter for the frontend, Dart Frog for the backend, and PostgreSQL for the database.

Core Features
-------------
* **Authentication**: Secure user registration and login.
* **Accounts & Budgets**: Create multiple accounts and set monthly budgets.
* **Transaction Tracking**: Log, categorize, and monitor spending.
* **Geolocation & Mapping**: Tag transactions with geographical locations using Google Maps and Google Places API.
